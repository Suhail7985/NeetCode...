<h3>Description</h3>
<p>
Given a binary tree, determine if it is <strong>height-balanced</strong>.
</p>

<p>
A binary tree is height-balanced if for every node, the difference in height
between the left and right subtree is at most 1.
</p>

<h3>Example 1:</h3>
<pre>
<strong>Input:</strong> root = [3,9,20,null,null,15,7]
<strong>Output:</strong> true
</pre>

<h3>Example 2:</h3>
<pre>
<strong>Input:</strong> root = [1,2,2,3,3,null,null,4,4]
<strong>Output:</strong> false
</pre>

<h3>Example 3:</h3>
<pre>
<strong>Input:</strong> root = []
<strong>Output:</strong> true
</pre>

<h3>Constraints:</h3>
<ul>
  <li>The number of nodes in the tree is in the range <code>[0, 5000]</code>.</li>
  <li><code>-10<sup>4</sup> &le; Node.val &le; 10<sup>4</sup></code></li>
</ul>
