    231. Power of Two

</h2>

<img src="https://img.shields.io/badge/Difficulty-Easy-brightgreen" alt="Difficulty: Easy" />
<hr>

<h3>Description</h3>
<p>
Given an integer <code>n</code>, return <code>true</code> if it is a power of two.
Otherwise, return <code>false</code>.
</p>
<p>
An integer <code>n</code> is a power of two if there exists an integer <code>x</code>
such that <code>n == 2<sup>x</sup></code>.
</p>

<h3>Example 1:</h3>
<pre>
<strong>Input:</strong> n = 1
<strong>Output:</strong> true
<strong>Explanation:</strong> 2<sup>0</sup> = 1
</pre>

<h3>Example 2:</h3>
<pre>
<strong>Input:</strong> n = 16
<strong>Output:</strong> true
<strong>Explanation:</strong> 2<sup>4</sup> = 16
</pre>

<h3>Example 3:</h3>
<pre>
<strong>Input:</strong> n = 3
<strong>Output:</strong> false
</pre>

<h3>Constraints:</h3>
<ul>
  <li><code>-2<sup>31</sup> &le; n &le; 2<sup>31</sup> - 1</code></li>
</ul>
