<h3>Description</h3>
<p>
An integer divisible by the sum of its digits is said to be a <strong>Harshad number</strong>.
You are given an integer <code>x</code>.
Return the sum of the digits of <code>x</code> if <code>x</code> is a Harshad number,
otherwise return <code>-1</code>.
</p>

<h3>Example 1:</h3>
<pre>
<strong>Input:</strong> x = 18
<strong>Output:</strong> 9
<strong>Explanation:</strong>
The sum of digits of x is 9.
18 is divisible by 9, so 18 is a Harshad number and the answer is 9.
</pre>

<h3>Example 2:</h3>
<pre>
<strong>Input:</strong> x = 23
<strong>Output:</strong> -1
<strong>Explanation:</strong>
The sum of digits of x is 5.
23 is not divisible by 5, so 23 is not a Harshad number and the answer is -1.
</pre>

<h3>Constraints:</h3>
<ul>
  <li><code>1 &le; x &le; 100</code></li>
</ul>
