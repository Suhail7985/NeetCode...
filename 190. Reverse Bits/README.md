<h3>Description</h3>
<p>
Reverse bits of a given 32-bit unsigned integer.
</p>

<h3>Example 1:</h3>
<pre>
<strong>Input:</strong> n = 43261596
<strong>Output:</strong> 964176192
<strong>Explanation:</strong>
Binary representation of 43261596:
00000010100101000001111010011100
After reversing:
00111001011110000010100101000000
Which equals 964176192.
</pre>

<h3>Example 2:</h3>
<pre>
<strong>Input:</strong> n = 2147483644
<strong>Output:</strong> 1073741822
<strong>Explanation:</strong>
Binary representation of 2147483644:
01111111111111111111111111111100
After reversing:
00111111111111111111111111111110
Which equals 1073741822.
</pre>

<h3>Constraints:</h3>
<ul>
  <li><code>0 &le; n &le; 2<sup>31</sup> - 2</code></li>
  <li><code>n</code> is even</li>
</ul>
